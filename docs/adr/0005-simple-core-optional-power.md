# 0005: Simple Core, Optional Power

## Status

Accepted

## Context

OKPF needs room for provenance, trust, evaluations, workflows, registries, and future cryptographic features, but the core format must remain approachable.

## Decision

A minimal OKPF pack should be writable by hand. Advanced capabilities are optional layers, not requirements for a valid basic pack.

## Consequences

Hello World remains simple, while serious interoperability and trust features can evolve without burdening every package.
