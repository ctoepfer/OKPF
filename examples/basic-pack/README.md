# Basic OKPF Example Pack

A minimal valid OKPF knowledge pack. This example demonstrates the required structure for a conformant pack.

## Contents

| File | Role | Description |
|------|------|-------------|
| `content/overview.md` | `guide` | Introduction to OKPF pack structure |

## Purpose

This pack exists to:

1. Give pack authors a copy-paste starting point
2. Serve as a test fixture for validators and tooling
3. Demonstrate the minimum required fields in `manifest.json`

## Validating this pack

From the repository root:

```bash
python reference/python/okpf_validate.py examples/basic-pack
```
