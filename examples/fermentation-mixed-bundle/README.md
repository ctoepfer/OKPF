# Fermentation Mixed Bundle Example

This example mirrors real multi-source training-pack needs while staying independent of any one consumer. It demonstrates beer, cider, mead, wine, and ingredient records with optional fermentation facets.

The records are synthetic or paraphrased examples. They are not full BJCP text and are not a replacement for BeerXML, BJCP, MeadXML, or other domain formats. OKPF can package those formats when licensing allows it.
