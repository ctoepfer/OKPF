## Prerequisites

Complete the Safety Precheck before beginning diagnostics.

## Step 1: Gather Reported Fault Information

- What symptom did the operator observe?
- When did the fault first appear?
- Was any unusual noise, smell, or visual indicator noted?
- Has this fault occurred before? If yes, what was the previous resolution?

## Step 2: Power-On Test (if safe to do so)

1. With the technician and operator clear of moving parts, briefly restore power.
2. Observe startup behavior: any error indicators, unusual sounds, or abnormal motion.
3. Power off immediately if any unsafe condition is observed.
4. Record observations.

## Step 3: Component-Level Inspection

Work through the following checks in order:

| Check | Pass Condition | Action if Fail |
|-------|----------------|---------------|
| Power supply connection | Secure, no damage | Re-seat or replace cable |
| Indicator LEDs or display | Expected state | Consult fault code table |
| Fan / ventilation | Unobstructed, rotating freely | Clear obstruction or replace fan |
| Mechanical linkages | No visible breakage or binding | Inspect further; do not force |
| Fluid levels (if applicable) | Within specified range | Top up per manufacturer spec (fictional) |

## Step 4: Fault Code Interpretation

If the equipment displays a fault code:

1. Record the code exactly as shown.
2. Look up the code in the fault symptom records included in this pack.
3. Follow the documented resolution steps.
4. If no matching code is found, escalate to level-2 support.

## Step 5: Document Findings

Before proceeding to repair:

- Record each check performed and its result.
- Note any components that appear worn or marginally functional even if not the primary fault.
- Take a photo of the fault condition if visible and safe to do so.
