# Original Prompt

Create a fictional all-grain West Coast IPA recipe for a homebrew-scale batch. Aim for a dry, crisp beer with firm bitterness, bright citrus/pine hop character, and moderate malt support.

Include target gravity, estimated bitterness, malt bill, hop schedule, yeast choice, and fermentation notes.
